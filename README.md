# Instruções da aplicação
### Instalar dependências
```
cd ng-cash-full-stack
npm run ambiente
```

### Iniciar Projeto com docker
```
cd ng-cash-full-stack
npm run compose:up
```

### Rodar Testes do Back-end
```
cd ng-cash-full-stack
npm run test-back
```
